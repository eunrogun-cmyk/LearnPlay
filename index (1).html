게임 업로드

당신의 게임을 세상에
공유하세요

ℹ️
GitHub Pages 방식으로 게임을 등록하세요
① 게임 HTML 파일을 GitHub 저장소에 업로드
② 아래에 파일명과 정보를 입력하면 게임 카드가 생성됩니다
③ 예: math-quiz.html
이용 방법

3단계로 시작하세요

01

게임 파일 준비

HTML, ZIP, APK 등 다양한 형식의 학습 게임 파일을 준비하세요. 웹 기반 게임이 가장 빠르게 공유됩니다.

02

정보 입력 & 업로드

게임 이름, 카테고리, 설명을 입력하고 파일을 업로드하면 자동으로 검토가 시작됩니다.

03

전 세계 학습자와 공유

승인 후 플랫폼에 게재되어 전 세계 학습자들이 즐길 수 있습니다. 플레이 통계도 확인하세요.
